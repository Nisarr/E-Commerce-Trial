# 📧 Google Apps Script — ইমেইল সেটআপ গাইড

এই ফোল্ডারে আপনার ব্যক্তিগত জিমেইল ব্যবহার করে PlayPen House এর জন্য একটি ফ্রি ইমেইল পাঠানোর সার্ভিস তৈরি করার সব প্রয়োজনীয় নির্দেশনা ও কোড দেওয়া আছে।

## 📁 এই ফোল্ডারের ফাইলসমূহ:

1. `Code.js` - এই ফাইলটিতে থাকা জাভাস্ক্রিপ্ট কোডটি আপনার Google Apps Script এ পেস্ট করতে হবে।
2. `ENV_VARS.md` - আপনার অ্যাপকে ইমেইল স্ক্রিপ্টের সাথে যুক্ত করতে কীভাবে এনভায়রনমেন্ট ভেরিয়েবলস (`.env` এবং `.dev.vars`) সেটআপ করবেন তার নির্দেশনা।

---

## 🚀 ধাপে ধাপে সেটআপ গাইড:

### ধাপ ১: Google Apps Script তৈরি করুন
1. প্রথমে [https://script.google.com](https://script.google.com) এ যান।
2. **"New Project"** এ ক্লিক করুন।
3. এডিটরে আগে থেকে থাকা সব লেখা মুছে ফেলুন।
4. এই ফোল্ডারের `Code.js` ফাইলটি ওপেন করে এর সব কোড কপি করে গুগল স্ক্রিপ্টের এডিটরে পেস্ট করুন। *(সরাসরি `.js` ফাইল থেকে কপি করার ফলে কোনো মার্কডাউন ফরম্যাটিং ভুলবশত কপি হওয়ার সম্ভাবনা থাকবে না!)*
5. **File → Save** এ ক্লিক করে প্রজেক্টের নাম দিন "PlayPen House Mailer"।

### ধাপ ২: Web App হিসেবে ডিপ্লয় (Deploy) করুন
1. **Deploy → New Deployment** এ ক্লিক করুন।
2. গিয়ার আইকনটিতে (⚙️) ক্লিক করে **"Web app"** সিলেক্ট করুন।
3. নিচের অপশনগুলো সেট করুন:
   - **Description**: "PlayPen House Email"
   - **Execute as**: **Me** (আপনার জিমেইল অ্যাকাউন্ট)
   - **Who has access**: **Anyone**
4. **Deploy** এ ক্লিক করুন।
5. কনফার্মেশনের জন্য পারমিশন চাইলে **Authorize access** এ ক্লিক করুন (এক্ষেত্রে "unverified app" ওয়ার্নিং আসলে 'Advanced' এ গিয়ে ক্লিক করুন)।
6. আপনার **Web App URL** টি কপি করে নিন (এটি দেখতে `https://script.google.com/macros/s/.../exec` এর মতো হবে)।

### ধাপ ৩: আপনার ওয়েব হুক নিরাপদ করুন
অন্য কেউ যেন আপনার এই URL ব্যবহার করে ইমেইল পাঠাতে না পারে তা নিশ্চিত করতে:
1. আপনার Google Apps Script এডিটরের বাম পাশের মেনু থেকে **Project Settings** (গিয়ার আইকন) এ যান।
2. নিচের দিকে স্ক্রল করে **Script Properties** এ যান এবং **Add script property** তে ক্লিক করুন।
3. **Property**: এর ঘরে লিখুন `WEBHOOK_SECRET`
4. **Value**: এর ঘরে যেকোনো একটি গোপন পাসওয়ার্ড দিন (যেমন: `my-custom-secret-key-123`) এবং সেভ করুন।

### ধাপ ৪: এনভায়রনমেন্ট ভেরিয়েবলস যোগ করুন
আপনার প্রজেক্টের কোন কোন ফাইলে এই ইউআরএল এবং সিক্রেটটি বসাতে হবে তা জানতে এই ফোল্ডারের `ENV_VARS.md` ফাইলটি ওপেন করুন।

### ধাপ ৫: পরীক্ষা (Testing) করুন
1. আপনার লোকাল সার্ভার চালু করুন: `npm run dev:full`
2. **Admin Panel → Settings** এ যান।
3. "Admin Notification Email" ফিল্ডে আপনার ইমেইলটি বসান।
4. **"Test"** বাটনে ক্লিক করুন — আপনার কাছে একটি টেস্ট ইমেইল চলে আসার কথা!
