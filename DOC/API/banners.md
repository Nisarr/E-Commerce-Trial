# Banners API

Endpoints for managing storefront banners.

## Endpoints

### List Banners
`GET /banners`

**Query Parameters:**
- `position` (string, optional): e.g., "hero", "sidebar".

---

### Create Banner (Admin)
`POST /banners`

---

### Update Banner (Admin)
`PATCH /banners/:id`

---

### Delete Banner (Admin)
`DELETE /banners/:id`
